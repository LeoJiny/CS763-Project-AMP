## Project Overview

This project studies whether ESM-2-based classifiers for antimicrobial peptides (AMPs) are
robust to small amino acid substitutions. We ask two questions:

1. Can a greedy adversarial attack (HotFlip-style) flip a correct AMP prediction with just 1–3 substitutions?
2. Can we formally certify robustness against such attacks using discrete randomized smoothing — without modifying the model?

## Notebooks

### `dataset_construction.ipynb`
Builds the binary AMP / non-AMP dataset from UniProt SwissProt.

- **Positive class (AMPs):** Reviewed UniProt entries annotated with keyword KW-0929 ("Antimicrobial"), length 10–100 AA — 2,518 sequences
- **Negative class (non-AMPs):** Reviewed UniProt entries excluding KW-0929, same length range — 2,400 sequences
- **Length-matching:** Non-AMPs are sampled bin-by-bin to match the AMP length distribution, preventing the classifier from learning length as a proxy for activity
- **Output:** `data/sequences.csv` (4,918 sequences total)

### `LR_classifier.ipynb`
Trains and evaluates the base AMP classifier on top of frozen ESM-2 embeddings.

- Loads ESM-2 (`facebook/esm2_t6_8M_UR50D`, 7.8M parameters) as a frozen feature extractor
- Extracts mean-pooled last-layer embeddings (320-dimensional) for all sequences
- Trains a logistic regression head on an 80/20 stratified train/test split
- Saves test embeddings, labels, and raw sequences (row-aligned) for downstream attack and defense notebooks
