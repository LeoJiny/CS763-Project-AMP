## Project Overview

This project studies whether ESM-2-based classifiers for antimicrobial peptides (AMPs) are
robust to small amino acid substitutions. We ask two questions:

1. Can a greedy adversarial attack (HotFlip-style) flip a correct AMP prediction with just 1–3 substitutions?
2. Can we formally certify robustness against such attacks using discrete randomized smoothing — without modifying the model?

## Notebooks

### `dataset_construction.ipynb`
Builds the binary AMP / non-AMP dataset from UniProt SwissProt.

- **Positive class (AMPs):** Reviewed UniProt entries annotated with keyword KW-0929 ("Antimicrobial"), length 10–100 AA.
- **Negative class (non-AMPs):** Reviewed UniProt entries excluding KW-0929, same length range.
- **Length-matching:** Non-AMPs are sampled bin-by-bin to match the AMP length distribution, preventing the classifier from learning length as a proxy for activity.
- **Output:** `data/sequences.csv`.

### `LR_classifier.ipynb`
Trains and evaluates the base AMP classifier on top of frozen ESM-2 embeddings.

- Loads ESM-2 (`facebook/esm2_t6_8M_UR50D`, 7.8M parameters) as a frozen feature extractor.
- Extracts mean-pooled last-layer embeddings (320-dim) for all sequences.
- Trains a logistic regression head on an 80/20 stratified train/test split.
- Saves test embeddings, labels, and raw sequences (row-aligned) for downstream notebooks.

### `attack.ipynb`
HotFlip-style greedy white-box attack on correctly classified test AMPs.

- At each step, enumerates all `19 × L` single-residue substitutions, scores them via ESM-2 + the classifier, and applies the substitution that most decreases P(AMP).
- Repeats up to `k_max = 3` substitutions or until the prediction flips.
- Reports attack success rate at k = 1, 2, 3 and the BLOSUM62 character of adversarial substitutions.

### `biovalidation.ipynb`
Filters flipped sequences for *biologically* AMP-like adversarial examples.

- Computes net charge, Boman index, and hydrophobic moment via `modlamp`.
- A *genuine* model failure is a flipped sequence that still passes all three AMP criteria.

### `certified_defense.ipynb`
Discrete randomized smoothing (Levine–Feizi / Ye et al. 2020) over the test set.

- For each sequence, draws T = 1,000 noisy copies (each position replaced uniformly at random with probability p ∈ {0.05, 0.10, 0.15}).
- Computes the Clopper–Pearson lower bound `p̲_A` on `P(AMP)` at 99.9% confidence.
- Certifies radius `R = floor(log p̲_A / log(1 − p))` when `p̲_A > 0.5`; otherwise abstains.
